# Project 4 - Terraform EC2 + Nginx

## Objective
Deploy a web server on AWS EC2 using Terraform.

## Resources created
- EC2 instance (Amazon Linux)
- Security Group (SSH + HTTP)
- Nginx installed automatically via user_data

## Result
Web server accessible via public IP:

http://44.198.171.23

## Commands used
- terraform init
- terraform plan
- terraform apply
