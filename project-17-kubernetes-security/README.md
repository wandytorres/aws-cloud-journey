# 🔐 Project 17 - Kubernetes Security

## Overview

This project demonstrates Kubernetes security best practices on Amazon EKS.

## Objectives

* Implement Role-Based Access Control (RBAC)
* Restrict pod communications using Network Policies
* Secure sensitive data using Kubernetes Secrets
* Prepare the cluster for runtime security with Falco
