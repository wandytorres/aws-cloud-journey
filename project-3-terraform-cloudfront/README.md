# Project 3 - Terraform S3 + CloudFront

## Objective
Provision a static website in AWS using Terraform with S3 and CloudFront.

## Resources created
- S3 bucket
- S3 website configuration
- public bucket policy
- index.html object
- CloudFront distribution

## Result
Website available through CloudFront:
https://d3l9nuu0paph7n.cloudfront.net

## Commands used
- terraform init
- terraform plan
- terraform apply
